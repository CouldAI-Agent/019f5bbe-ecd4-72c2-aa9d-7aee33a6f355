# Medication Return Tracker

A Flutter application designed to help pharmacy staff carefully track and document medications returned from hospital wards. This ensures accurate inventory management and proper accountability, especially before stock take periods.

## Features

- **Dashboard:** View all logged medication returns with their status, quantity, and remarks.
- **Add Return Form:** Safely log a new medication return.
- **Variance Tracking:** Require staff to provide clear remarks when there is a quantity difference.
- **Staff Nurse Logging:** Record the responsible staff nurse's name for items being pushed to the floor.

## Getting Started

1. Ensure you have Flutter installed.
2. Run `flutter pub get` to fetch dependencies.
3. Start the application with `flutter run`.

## Tech Stack

- Framework: Flutter
- State Management: Provider
- Routing: Standard Flutter Navigation
